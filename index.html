Passos da nossa primeira atividade

  1. Criar uma conta no github
  2. Criar um repositório do github pages - o nome do repositório DEVE ser seulogin.github.io
  3. Clone o repositório para o seu git local. Você precisa ter o git instalado (linux/Unix) ou um cliente no Windows (gitbash, por exemplo)
  4. Crie um arquivo index.html no seu repositório
  5. git add no seu index.html
  6. git commit -m "mensagem"
  7. Envie as alterações para o repositório remoto - git push origin main
  8. me mande o seu nome e endereço do seu site no chat para que eu crie a página da turma
